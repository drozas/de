\section{Introducción}
\noindent
En \LaTeX{} se pueden utilizar ficheros de estilos (con extensiones \verb|.sty|
y \verb|.cls|), los cuales contienen configuraciones específicas para cada tipo
de documento. En esta prática se va a utilizar un fichero llamado
\verb|practica6.sty|, que contiene el estilo para el documento a crear.\par
Se importará en el preámbulo (\verb|usepackage{practica6}|) 
Para especificar el título se debe utilizar la orden \verb|\title{titulo}|
antes del comienzo del contenido del documento.
Para indicar el resumen se usará el entorno \verb|\begin{abstract}| \ldots
\verb|\end{abstract}|.
